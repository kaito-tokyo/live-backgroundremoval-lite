# Live Background Removal Lite



## How to Install



1. **Open** *C:/ProgramData* (*%PROGRAMDATA%*)
2. **Create subfolders**:
*C:/ProgramData/obs-studio
C:/ProgramData/obs-studio/plugins*
3. **Install**:
Place *live-backgroundremoval-lite* folder into
*C:/ProgramData/obs-studio/plugins/live-backgroundremoval-lite*
4. **Check:**
*C:/ProgramData/obs-studio/plugins/live-backgroundremoval-lite/bin/64bit/live-backgroundremoval-lite.dll*



## How to Uninstall



The plugin binary is installed under *%ProgramData%/obs-studio/plugins/live-backgroundremoval-lite*. You can remove this plugin from your OBS installation by removing this *live-backgroundremoval-lite* folder.



## Supported environments

Standalone OBS installations (for example, installed under *C:/Program Files* or *D:/*) are supported. We don't provide any help for portable installations. Do not ask about portable installation on our project page.
